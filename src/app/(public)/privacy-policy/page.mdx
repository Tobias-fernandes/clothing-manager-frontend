# Política de Privacidade

**Última atualização:** maio de 2026

O **Clothing Manager** respeita a privacidade dos seus usuários e está comprometido com a proteção dos dados pessoais em conformidade com a **Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018)**.

---

## 1. Quais dados coletamos

### Dados fornecidos pelo usuário
- Nome completo
- Endereço de e-mail
- Senha (armazenada exclusivamente em formato de hash — nunca em texto puro)
- Perfil de acesso (gerente ou funcionário)

### Dados gerados pelo uso do sistema
- Registros de vendas realizadas
- Movimentações de estoque
- Entradas e saídas financeiras
- Data, horário e identificação do usuário responsável por cada ação

### Dados técnicos
- Endereço IP de acesso
- Tipo de dispositivo e navegador
- Cookies de sessão e autenticação

---

## 2. Como utilizamos os dados

Os dados coletados são utilizados exclusivamente para:

- Autenticar e autorizar o acesso dos usuários ao sistema.
- Registrar e rastrear as operações realizadas na plataforma.
- Garantir a integridade dos dados de estoque, vendas e financeiro.
- Manter o histórico de ações para fins de auditoria interna da loja.
- Melhorar a experiência e o desempenho do sistema.

**Não utilizamos os seus dados para fins publicitários. O Clothing Manager não exibe anúncios e não compartilha dados com anunciantes.**

---

## 3. Compartilhamento de dados

O Clothing Manager **não vende, aluga ou compartilha** dados pessoais com terceiros, exceto nas seguintes situações:

- **Prestadores de serviço de infraestrutura** — como servidores e banco de dados em nuvem, que atuam como operadores de dados sob contrato de confidencialidade.
- **Obrigação legal** — quando exigido por autoridade competente ou determinação judicial.

---

## 4. Armazenamento e segurança

- Os dados são armazenados em banco de dados **PostgreSQL** com acesso restrito e autenticado.
- As senhas são protegidas com criptografia **bcrypt** e nunca são armazenadas em texto puro.
- O acesso à plataforma é protegido por autenticação via **JWT (JSON Web Token)**.
- Os cookies de sessão utilizam as flags **HttpOnly**, **Secure** e **SameSite** para proteção contra ataques.
- Adotamos boas práticas de segurança no desenvolvimento para proteger os dados contra acesso não autorizado.

---

## 5. Cookies

O Clothing Manager utiliza cookies estritamente necessários para o funcionamento do sistema:

| Cookie | Finalidade | Duração |
|---|---|---|
| `token` | Autenticação do usuário logado | 8 horas |

Não utilizamos cookies de rastreamento, analytics ou publicidade.

---

## 6. Retenção de dados

Os dados são mantidos pelo período necessário para o funcionamento do sistema e para fins de auditoria interna da loja. Dados de usuários inativados são preservados para manter a integridade do histórico de operações.

---

## 7. Direitos do titular

Em conformidade com a LGPD, o usuário tem direito a:

- **Acesso** — solicitar cópia dos dados pessoais armazenados.
- **Correção** — solicitar a atualização de dados incorretos.
- **Exclusão** — solicitar a remoção dos dados, quando legalmente possível.
- **Portabilidade** — solicitar os dados em formato estruturado.
- **Revogação do consentimento** — retirar o consentimento a qualquer momento.

Para exercer esses direitos, entre em contato com o suporte do Clothing Manager.

---

## 8. Menores de idade

O Clothing Manager é um sistema de uso exclusivamente profissional e não é destinado a menores de 18 anos.

---

## 9. Alterações nesta política

Esta política poderá ser atualizada periodicamente. Em caso de alterações relevantes, os usuários serão notificados pelo sistema. O uso continuado da plataforma após a notificação implica na aceitação da política atualizada.

---

## 10. Contato

Para dúvidas, solicitações ou reclamações relacionadas à privacidade dos seus dados, entre em contato com o suporte do Clothing Manager.
